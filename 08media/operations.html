Operations

Your camera has many modes on it. These options tell the camera how to set the shutter speed and aperature, adjust ISO, set white balance, pop the built-in flash, or change other internal camera settings.


Symbol Camera Mode Discription
AUTO FULL AUTOMATIC MODE Camera sets all of the settings automatically for a properly exposed and sharp image.
M MANUAL MODE Photographer sets the shutter speed, aperature, and ISO. Camera provids a meter reading.
S/Tv SHUTTER PRIORITY MODE You set shutter speed, camera sets proper aperature. ISO is set seperately. Exposure adjusted through the exposure compensation setting.
A/Av APERATURE PRIORITY MODE You set aperature, camera sets proper shutter speed. ISO set seperately. Exposure adjusted through the exposure compensation setting.
P PROGRAM MODE Camera pairs up an aperature and shutter speed combination. ISO set seperately. Exposure adjusted through the exposure compensation setting.
FLOWER MACRO/CLOSE-UP MODE Camera sets the ISO, shutter speed, and aperature to aid in macro photography.
MOUNTAIN LANDSCAPE MODE Camera sets the ISO, shutter speed, and aperature to aid in macro photography. Built-in flash is turned off.
PERSON WITH MOON/STAR NIGHT PORTRAIT MODE Camera combines the built-in flash and slow shutter speed.
PERSON PORTRAIT MODE Camera sets a large aperature to blur background, and overrides other settings.
PERSON RUNNING SPORTS MODE Camera sets a fast shutter speed to freeze action.
AF-A AUTO-SERVO AF Gives camera control on whether to select AF-S or AF-C. Use with many focus points.
AF-S SINGLE-SERVO AF When shutter-release button is pressed half way down the focus is locked. Use with stationary objects.
AF-C CONTINUOUS-SERVO AF When shutter-release button is pressed half way down the camera focuses continuously. Use with moving subjects.
